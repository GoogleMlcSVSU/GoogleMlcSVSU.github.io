Introduction

Google Developers Student Club-SVSU Google Developer Student Clubs are community groups for college and university students interested in Google developer technologies. Students from all undergraduate or graduate programs with an interest in growing as a developer are welcome. By joining a GDSC, students grow their knowledge in a peer-to-peer learning environment and build solutions for local businesses and their community. Google Developer Student Clubs is a program supported by Google Developers.

How do I get involved? You can start your own project or jump onto a team for an existing project - or just visit us for events and workshops. Check #resources for a list of ongoing projects and to learn more about starting your own.

Machine Learning Competetion 2022

Here is the registration form link for the MLC Competition next Friday (December 09)! Google register form You can compete individually, or with a team up to 3.
If you are planning on competing, please all register individually using the SAME TEAM NAME The contest will have three questions: two regression problems, and one numeric classification problem.
Prizes (Gift Cards) will be awarded to the team with the best evaluation metrics: (high accuracy) for classification and low validation loss for regression (making sure there are no high amounts of overfitting). Also, there is a list of professors and classes offering extra credit! The list can be found in the registration form.
Please make sure to register by 5:00 PM Thursday, December 08!!
Any questions, please contact Mason, Manish , or Professor Aos! Thanks!

Resource for the Comp:

  1. 1st of all, start with setting up libraries i.e. import the necessary libraries
  2. 2nd, load the dataset using pandas library
  3. 3rd, split the dataset into training and testing set
  4. 4th, create a model using keras library
  5. 5th, compile the model
  6. 6th, train the model
  7. 7th, evaluate the model
